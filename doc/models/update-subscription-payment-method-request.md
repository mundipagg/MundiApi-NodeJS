
# Update Subscription Payment Method Request

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |

| `cardId` | `string` | Required | Card id |
| `card` | [`CreateCardRequest`](../../doc/models/create-card-request.md) | Required | Card data |
